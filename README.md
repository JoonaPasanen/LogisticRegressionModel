A simple logistic regression model that uses age differentials, ATP-ranking differentials and surface specific elos from the last 20-years of ATP matches to predict the probabilities that each of the players of a match is the winner. Overall the model performs comparably to publicly found models of similar complexity.

Metrics from the model:

Log loss: 0.6147478597767255

Brier score: 0.2139489555484533

Accuracy: 0.6544554455445545
